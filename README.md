# 오늘한입 스마트폰 단독 PWA`n`n이 버전은 Python 서버 없이 동작합니다. 기록·설정·압축 사진은 스마트폰 브라우저의 IndexedDB에 저장됩니다.`n`n## 무료 설치 방법`n`n1. GitHub에서 무료 계정을 만들고 새 공개 저장소를 만듭니다.`n2. 이 폴더의 파일 전체를 저장소 최상위에 업로드합니다.`n3. 저장소 Settings → Pages에서 Deploy from a branch, main, /(root)를 선택합니다.`n4. 생성된 https 주소를 스마트폰에서 엽니다.`n5. Android Chrome 메뉴 또는 iPhone Safari 공유 메뉴에서 홈 화면에 추가를 선택합니다.`n6. 첫 접속 후에는 오프라인에서도 실행할 수 있습니다.`n`n## 중요한 백업 안내`n`n데이터는 해당 스마트폰과 브라우저에만 있습니다. 앱 삭제, 사이트 데이터 삭제, 기기 분실에 대비해 설정 화면에서 JSON 백업과 CSV 내보내기를 정기적으로 저장하세요. 현재 JSON 백업은 기록 데이터 중심이며 식사 사진 파일은 포함하지 않습니다.`n`n앱 업데이트 전에도 반드시 백업하세요. X/Twitter에는 자동 게시하지 않으며 요약문을 복사해 직접 게시합니다.
